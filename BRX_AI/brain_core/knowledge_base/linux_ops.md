# Base de Conhecimento Técnico - BRX AI

## Comandos de Sistema (Arch Linux)
- Atualização: `sudo pacman -Syu`
- Instalação: `sudo pacman -S [pacote]`
- Limpeza de Cache: `sudo pacman -Sc`
- Logs do Sistema: `journalctl -xe`

## Gerenciamento de Processos
- Listar: `ps aux`
- Matar: `kill -9 [PID]`
- Monitorar: `top` ou `htop`

## Estrutura do Projeto BRX AI
- Localização: `/opt/brx_ai_app` ou diretório do repositório.
- Configurações: `~/.brx_ai/`
- Logs: `~/.brx_ai/logs/brx_ai.log`
